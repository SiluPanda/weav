A single Rust binary that gives AI applications sub-millisecond access to a context graph — combining knowledge graph storage, vector search, full-text retrieval, and token-aware context assembly in one process.

05 — Architecture

Designed for in-memory speed

Single-Process, Multi-Protocol

  • Rust — no GC pauses, no JVM overhead, predictable p99 latency
  • Thread-per-core with keyspace sharding. Per-graph RwLocks for concurrent reads
  • String interning — labels and property keys as u16 IDs, not heap strings
  • SmallVec adjacency — first 8 neighbors stack-allocated
  • 11 modular crates with zero cyclic dependencies
